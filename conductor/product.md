# Product Definition: AeroCMS

## Vision
AeroCMS is a high-performance, modular content management system built on the cutting edge of the .NET ecosystem (.NET 10.0+). It aims to provide a highly flexible, extensible, and developer-friendly platform for building modern web applications, from simple blogs to complex enterprise systems, with native support for AI, real-time communication, and distributed architectures.

## Target Audience
- **Developers:** Seeking a robust, modular framework to build custom CMS solutions.
- **Content Creators:** Requiring a powerful and intuitive interface for managing multi-channel content.
- **Enterprise Organizations:** Needing a scalable, observable, and secure platform for large-scale digital experiences.

## Core Pillars
1. **Modular Architecture:** A plug-and-play system where features (AI, Blog, CRM, etc.) are isolated, testable, and composable.
2. **Developer Experience (DX):** Leveraging the latest .NET features, TDD, and strong typing to ensure high-quality code and rapid iteration.
3. **Performance & Scalability:** Built for high-throughput scenarios using Marten, Wolverine, and advanced caching strategies (FusionCache, Garnet).
4. **AI-First:** Deeply integrated AI capabilities for content generation, assistance, and automation.

## Key Features (MVP & Beyond)
- **Advanced Content Modeling:** Flexible schema-less storage with Marten.
- **Robust Identity Management:** Integrated authentication and authorization (JWT, OpenID).
- **Extensive Module Library:** Out-of-the-box modules for SEO, Blog, Forms, Chat, and more.
- **Distributed Ready:** Orchestration via .NET Aspire and messaging via Wolverine.
- **Real-time Engine:** Integrated support for SignalR and MagicOnion (gRPC).
- **Rich UI:** Blazor-based management interface with Radzen/Radzen.Theming.
